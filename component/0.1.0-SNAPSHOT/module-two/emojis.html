Emojis

asciidoctor-emoji is an extension for Asciidoctor that turns emoji:heart[] into !

Additionally you can configure the size of an emoji. By default, the size is 24px but you can use one of the following:

  • 1x (17px)

  • lg (24px)

  • 2x (34px)

  • 3x (50px)

  • 4x (68px)

  • 5x (85px)
